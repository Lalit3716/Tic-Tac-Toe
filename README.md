# Tic Tac Toe With AI
## Getting Started
#### **Download the Zip File**, and extract it anywhere. After extracting place all files in a single folder and open game.pyw. 
## Description Of Game
<ul>
  <li>This is a good old game Of Tic Tac Toe</li>
